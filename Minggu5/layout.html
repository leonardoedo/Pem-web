Pengertian Layout

HTML layout menyediakan cara untuk mengatur halaman web dengan cara yang baik,
terstruktur dengan baik, dan dalam bentuk responsif atau kita dapat mengatakan bahwa
HTML menentukan cara di mana halaman web dapat diatur. Layout halaman web bekerja dengan
pengaturan elemen visual dari dokumen HTML.
